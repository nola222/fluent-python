# 为什么len不是普通方法 实用剩余纯粹
```python
# CPython会直接从一个C结构体里面读取对象的长度，完全不会调用任何方法。
# 为了让Python自带的数据结构可以走后门，abs也同理。
# 把len用于自定义数据类型。这种处理方式在保持内置类型的效率和保证语言的一致性之间找到了一个平衡点，也印证Python之禅里面的一句话：“不能让特例特殊到开始破坏既定规则。”
# abs()和len()看作一元运算符的话，虽然看起来像面向对象语言中的函数，但实际上又不是。ABC语言是Python的直系祖先，内置一个#运算符，#s作用类似于len。
# x#s这样的中缀运算符，作用是计算s中x的出现的次数。相当于Python中s.count(x)，s是一个序列类型。
```
